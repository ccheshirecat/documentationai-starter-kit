<Update label="2024-11-01" description="v1.3.0" tags={["feature", "improvement"]}>

## New Features

- Added support for custom brand colors like `#3B82F6` for HYPR themes
- Introduced `<Video>` component for embedding tutorials directly in docs
- Enabled real-time preview for MDX edits in the dashboard

## Improvements

- Faster page rendering with optimized Mermaid diagram parsing
- Better mobile responsiveness for `<Tabs>` and `<Steps>` components

## Bug Fixes

- Fixed parsing errors with nested code blocks using 4 backticks
- Resolved issues with special character escaping in prose (`{API_KEY}` now displays correctly)

</Update>

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "bugfix"]}>

## New Features

- New `<Columns>` layout with up to 5 columns for feature grids
- Integrated Lucide icons across all components (e.g., `icon="rocket"`)
- Added `default-open` prop to `<Expandable>` for better UX

## Bug Fixes

- Corrected H1 heading restrictions—pages now enforce H2 starts
- Patched JSX attribute parsing for double quotes only (`title="Text"`)

</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["breaking", "feature"]}>

## Breaking Changes

- Removed `<Mermaid>` JSX component—use code blocks only (````markdown ```mermaid graph TD; A-->B ``` ````)

## New Features

- `<Request>` and `<Response>` components for API docs
- `<ParamField>` for precise parameter documentation

## Improvements

- Enhanced frontmatter validation—no blank lines allowed

</Update>
